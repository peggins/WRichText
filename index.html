WRich-Text: Web Text editor with rich-text formatting

This is a free, open-source project. A simple web project in the form of a text editor with RIch-Text style formatting. The project can be installed on your machine by accessing the project's official repository (just click here). Below you will see a brief list of the formatting styles we currently have.

  • Link
  • Bold
  • Italic
  • Underline

Why use WRich-Text?

One of the main reasons for using WRich-Text is that it's easy to access via the Internet, and can be used on both computers and mobile devices. You'd also like to use it because it doesn't require you to install any packages on your operating system, your browser will take care of everything for you. Another highlight is its lightness - by using primary technologies and without a ready-made structure or too many libraries to cause bloat in the application, you enjoy a simple, light and straightforward application.
